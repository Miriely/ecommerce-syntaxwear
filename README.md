# SyntaxWear - E-commerce de Tênis e Sneakers

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

A **SyntaxWear** é uma landing page de e-commerce moderna e responsiva, focada na venda de tênis e sneakers exclusivos. O projeto foi desenvolvido com foco em performance, organização modular de estilos e uma interface de usuário limpa e atraente.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- **Menu Mobile:** Navegação intuitiva com menu lateral (drawer) para telas menores.
- **Seção Hero:** Destaque para produtos principais com chamadas para ação (CTA).
- **Categorias:** Navegação rápida por estilos (Casual, Esporte, Moderno e Futurista).
- **Grade de Produtos:** Exibição organizada de modelos em destaque.
- **Newsletter:** Área de captura de e-mails integrada ao rodapé.
- **Social Media:** Links rápidos para as principais redes sociais.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web fundamentais, priorizando a organização do código:

- **HTML5:** Estrutura semântica para melhor SEO e acessibilidade.
- **CSS3:** Estilização avançada utilizando:
    - **Variáveis CSS:** Centralização de cores e fontes para fácil manutenção.
    - **Arquitetura Modular:** Divisão de estilos por componentes (Header, Footer, Hero, etc.).
    - **Flexbox:** Sistema de layout flexível e moderno.
    - **Media Queries:** Garantia de responsividade.
- **Google Fonts:** Utilização da fonte 'Ubuntu' para uma tipografia moderna.
